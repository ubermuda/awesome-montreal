# Awesome Montreal

## Breakfast & coffee

* L'avenue (brunch)
* Café Névé (café, cookies, etc)

## Others

* Bilboquet (glacier)
* La réserve du comptoir (boucherie, charcuterie)

* La banquise (poutine)
* Poutineville (poutine)
* Ma Poule Mouillée (rotisserie portugaise)
* L'arepera du plateau (venezuélien)

* Bagels Saint Viateur (bagels)
